# My-Animals-10-Model

## How to Run
* If needed, run `pip install tensorflow seaborn`.
* Unzip the Animals-10 folder found at https://www.kaggle.com/datasets/alessiocorrado99/animals10.
* In `animals_cnn.py`, configure the `dataset_path` to point at the folder containing all the animal folders.
* Run `python animals_cnn.py`.
